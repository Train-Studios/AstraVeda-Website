# AstraVeda Website

Production-ready marketing and legal website for AstraVeda. This app is isolated from the mobile app and lives entirely inside `website/`.

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Vercel-ready deployment

## Local Development

```bash
cd website
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production Build

```bash
cd website
npm run build
npm run start
```

## Vercel Deployment

1. Import the repository into Vercel.
2. Set the project root to `website`.
3. Use the default Vercel build command:

```bash
npm run build
```

4. Use the default output generated by Next.js.

## Environment Variables

The website works without environment variables. Optional:

```bash
NEXT_PUBLIC_SITE_URL=https://your-domain.example
```

Use this to generate canonical URLs, sitemap URLs, and robots.txt references for your production domain.

## Legal Content

Legal markdown files live in:

```text
content/legal/privacy_policy.md
content/legal/terms.md
```

Update those files to revise the public legal pages.

## Notes

- The website does not affect the React Native mobile app.
- No private backend secrets should be placed in this website.
- Support email is configured in `lib/site.ts`.
